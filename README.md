# Diabetes Prediction App

This is a Machine Learning web app built using Streamlit.

## About
It predicts whether a person is diabetic based on medical input features using Logistic Regression.

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas
- NumPy

## How it works
User enters health details → model processes data → prediction is shown (Diabetic / Not Diabetic)

## Model
Trained using Logistic Regression on Pima Indians Diabetes dataset.
